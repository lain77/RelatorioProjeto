Esta seção contém a documentação técnica, a descrição do projeto e os resultados do projeto **OXCART PROGRAM**.

DESCRIÇÃO DO PROJETO

O **OXCART PROGRAM** é um aplicativo móvel inovador que atua como um **banco de dados interativo de aeronaves**. Seu principal diferencial é a gamificação do aprendizado: o usuário cria uma conta e participa de **quizzes** temáticos sobre aviação. Ao acumular pontos com acertos, o usuário é recompensado com o **desbloqueio de novos modelos de aeronaves** e acesso a informações detalhadas, incentivando a exploração contínua do banco de dados.
